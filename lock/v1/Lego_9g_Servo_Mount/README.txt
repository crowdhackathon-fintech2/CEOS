# Summary

A LEGO compatible mount for the SG-9 or 9G micro servo. The mounting is offset to allow for the axles to be horizontally centered. Also, the mount is designed to be vertically centered with LEGO gears.

# Print Settings

Printer: MP Select Mini
Rafts: No
Supports: No
Resolution: 0.1mm
Infill: 0%

Notes: 
Infill can be ZERO since all LEGOs are hollow. This is designed to print without under supports. If you do print supports or a brim then they will be a pain to remove. If your bed is not level you will get a lip on the inner tube, "elephant feet." Make sure you have good bed adhesion since the is very little contact initially.
